Função quadrática Função Modular

Conceito

A função quadrática é uma função f = IR -> IR, que possui como fórmula: f(x) : ax² + bx + c

Existem algumas regras para que a função seja quadrática, entre elas:

  • O coeficiente A precisa ser diferente de 0;
  • O gráfico é uma parábola;
  • O domínio é D(f) = IR e a imagem depende da concavidade (negativa ou positiva) e do vértice;

Para calcular o f(x) dado o x basta substituir o valor de x na função. Para calcular o valor de x ou as raízes da função é necessário realizar o cálculo de Bháskara. Existem técnicas, a partir do valor do delta, para saber quantas raízes a função irá possuir:

  • Δ = 0 : possui 2 raízes reais iguais, então o eixo x do gráfico terá uma única intersecção;
  • Δ > 0 : possui 2 raízes reais diferentes, então o eixo x do gráfico terá duas intersecções;
  • Δ < 0 : não possui raízes reais, então o eixo x não possuirá nenhuma intersecção.

A construção do gráfico pode ser feita a partir de alguns passos:

  1. Identificar a concavidade do gráfico:
    • se coeficiente a > 0 a concavidade será para cima
    • se coeficiente a < 0 a concavidade será para baixo
  2. Calcular as raízes da função;
  3. Calcular o corte no eixo y, atribuindo o valor de x = 0 na função (o corte no eixo y é igual ao valor do coeficiente C);
  4. Calcular as coordenadas do vértice;
