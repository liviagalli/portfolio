Conceito

Para entender os conceitos de PA e PG, primeiro precisamos estudar sobre as sequências numéricas. Por conceito, uma sequência numérica é a sucessão de números que seguem um padrão específico, sendo uma função que associa, a cada elemento n ∈ IN*, um único elemento an ∈ IR.

f: IN* -> IR, onde IN* corresponde às posições dos termos da sequência.


Pela Lei de Formação, podemos estabelecer fórmulas diferentes para encontrar os termos da sequência. As mais utilizadas são:

  • Fórmula de recorrência: são dadas duas regras, uma para estabelecer o primeiro termo e outra para calcular os demais termos a partir de seu antecessor. Exemplo:

    a1 = 0

    an = an - 1 + 2

    Sequência = {0, 2, 4, 6, 8, 10, …}

  • Termo geral: dada uma fórmula que expressa an em função de n. Exemplo:

    Sendo n ≥ 1 --> an = 2n - 2

    Sequência = {0, 2, 4, 6, 8, 10, …}


Progressão Aritmética é a sequência de números em que cada termo, a partir do segundo, é o resultado da soma de seu antecessor com uma constante r (razão). Ou seja, a subtração de um termo an com seu antecessor (a partir do segundo termo) resultará na razão.

(1, 3, 5, 7, 9, …) ----> a1 = 1 e r = 2


A PA pode ser classificada em:

  • Crescente: r > 0;
  • Constante: r = 0;
  • Decrescente: r < 0;

O termo geral da PA pode ser definido como: an = a1 + (n - 1) * r

onde:

  • a1 = primeiro termo
  • an = termo geral
  • n = posição do termo
  • r = razão

O gráfico de uma PA é semelhante ao crescimento linear de uma função afim, entretanto, ele não pode ser uma reta pois seu domínio são apenas os números naturais maiores que 0. Sendo assim, o gráfico da PA é constituído de pontos alinhados.


A interpolação aritmética consiste em inserir k meios aritméticos entre dois termos a e b. Para isso, calculamos quantas posições a PA terá a partir da equação: n = k + 2, e depois aplicamos o termo geral para descobrir a razão.


Para somar todos os termos da PA, usamos a fórmula: Sn = (a1 + an) / 2



Progressão Geométrica é a sequência de números onde cada termo, a partir do segundo, é o produto do seu antecessor com uma constante q. Ou seja, a divisão de um termo an com seu antecessor (a partir do segundo termo) resultará na razão.

(-1, -2, -4, -8, -16…) ----> a1 = -1 e q = 2.


Podemos classificar a PG em:

  • Crescente:
    • PG com termos positivos: q > 1;
    • PG com termos negativos: 0 < q < 1;
  • Constante:
    • PG com todos os termos nulos: a1 = 0 e q qualquer;
    • PG com termos iguais e não nulos: a ≠ 0 e q = 1;
  • Decrescente:
    • PG com termos positivos: 0 < q < 1;
    • PG com termos negativos: q > 1;
  • Alternantes: cada termo tem sinal contrário ao do termo antecessor. Isso ocorre quando q < 0.
  • Estacionárias: apenas o primeiro termo é diferente de 0. Nesse caso, a1 ≠ 0 e q = 0.

O termo geral da PG é definido como: an = a1 * qn-1


O gráfico da PG é semelhante ao crescimento exponencial, entretanto, por causa da restrição do seu domínio, não é possível traçar a linha curva. Dessa forma, o gráfico será constituído por pontos que formam uma curva exponencial.


A interpolação geométrica segue o mesmo princípio da interpolação aritmética, sendo diferente apenas na fórmula para calcular a razão da PG.


A soma dos termos de uma PG finita é dada por: Sn = a1 * [(1 - q n) / (1 - q)]


A soma dos termos de uma PG infinita, quando -1 < q < 1 e com limite igual a zero, é dada por: limSn = a1 / 1 - q
