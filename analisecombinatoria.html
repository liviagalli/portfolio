Permutação


A palavra “permutar” possui o sentido de trocar de posição, ou seja, em análise combinatória é associada ao conceito de trocar de lugar os elementos de um problema combinatório. A permutação considera a ordenação dos elementos, além de se dividir em duas: permutação simples e permutação com repetição.


Permutação Simples


A permutação simples é a ordenação de elementos de um conjunto, onde nenhum deles se repete. Indicamos por Pn o número de permutações simples de n elementos:

Pn = n * (n-1) * (n-2)..... 3 * 2 * 1


Percebemos que nessa multiplicação que os fatores são n e inteiros menores do que ele, por isso chamamos de fatorial. Então, a fórmula de permutação simples pode ser entendida como:

Pn = n!

Permutação com Repetição


A permutação com repetição é a ordenação de elementos de um conjunto, porém esses elementos podem se repetir e, por isso, a permutação é diferente. A fórmula para a permutação com repetição é:

Pna,b,c... = n! / a! * b! * c! …


Nessa fórmula, podemos interpretar que n! é a quantidade de elementos do conjunto e os coeficientes a, b e c são os elementos (quantidade deles) que se repetem dentro do conjunto. A permutação entre elementos repetidos pode gerar resultados iguais, o que pode interferir na contagem dos diferentes tipos de ordenação que um conjunto possui.


Exemplo:


Diferenciando Permutação Simples de Permutação com Repetição
Quantos são os anagramas da palavra LUCIDEZ?
Nesse caso, a palavra “LUCIDEZ” não possui nenhuma letra repetida em seu conjunto, então podemos utilizar a permutação simples.
Pn = n!
P7 = 7!
P7 = 7 * 6 * 5 * 4* 3 * 2 * 1
P7 = 5040 anagramas
Quantos são os anagramas da palavra BANANA?
Nesse caso, a palavra “BANANA” repete 3 vezes a letra A e 2 vezes a letra N, então precisamos utilizar a permutação com repetição
P63, 2 = 6! / 3! * 2!
P63, 2 = 6 * 5 * 4 / 2
P63, 2 = 120 / 2
P63, 2 = 60 anagramas

Combinação Simples


Assim como permutação, combinação simples é um método utilizado para organizar um conjunto de elementos. A diferença entre os dois métodos é que na combinação simples a ordem dos elementos do conjunto não interfere, porém não há repetição no conjunto. A expressão geral de combinação simples é dada por:

Cn, p = n! / p! * (n - p)!

onde n é a quantidade total de elementos não repetidos e p é a quantidade de elementos distintos em cada agrupamento.
