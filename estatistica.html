Conceito

A estatística compreende o conjunto de métodos utilizados para coletar e interpretar dados, visando possibilitar a previsão de fenômenos futuros a partir de cálculos. Estudamos a estatística descritiva, que descreve, organiza e apresenta os dados através de tabelas, gráficos e medidas.


Nesse aspecto, temos alguns conceitos para serem entendidos:

  • População: conjunto de elementos que irão ser investigados.
    • censo ou recenseamento;
    • são feitos, geralmente, quando a população é menor e/ou quando se exige um resultado exato.
  • Amostra: é uma parte da população.
    • pesquisa por amostragem;
    • são feitas, geralmente, quando opta-se pela economia de tempo e/ou de recurso financeiro, quando a população é muito grande ou infinita e/ou quando são feitos testes destrutivos.

Para a organização dos dados, são construídas tabelas constituídas de título, cabeçalho, linhas e rodapé. Essas tabelas podem trazer dados absolutos (contagem real) e dados relativos (em porcentagem).


Em alguns casos, necessita-se dividir a tabela em classes para torná-la mais enxuta ou mais organizada. Existe alguns passos para organizar as classes da tabela:

  1. Para saber o número mais adequado de classes, faz-se a raiz quadrada do número total de objetos pesquisados.
  2. Para saber a amplitude de cada classe, dividimos a amplitude total (diferença entre a maior e a menor idade) pelo número de classes.
  3. Para uma melhor organização, geralmente os resultados são arredondados conforme a necessidade.

A construção do gráfico pode ser feita a partir de alguns passos:

  1. Identificar a concavidade do gráfico:
    • se coeficiente a > 0 a concavidade será para cima
    • se coeficiente a < 0 a concavidade será para baixo
  2. Calcular as raízes da função;
  3. Calcular o corte no eixo y, atribuindo o valor de x = 0 na função (o corte no eixo y é igual ao valor do coeficiente C);
  4. Calcular as coordenadas do vértice;

Para estabelecer melhor a exibição da evolução dos dados, são construídos gráficos. Esses gráficos podem ser em colunas, em barras, em setores, em curvas ou em histogramas.

Em estatística, possuímos as medidas de posição, sendo elas:

  • Média Aritmética Simples: é o quociente entre a soma de todos os elementos do conjunto e a quantidade de elementos nesse conjunto.

    Ms = (x1 + x2 + x3 + … + xn) / n

    onde x são os termos e n é a quantidade de termos ( n > 1 )

  • Média Aritmética Ponderada: usada quando há valores que aparecem mais vezes no conjunto dado ou valores que possuem pesos diferentes.

    Mp = (P1 + N1) + (P2 + N2) + (P3 + N3) / (P1 + P2 + P3)

  • Mediana: na ordem crescente, a mediana é o valor que divide a amostra ou população em duas partes.
    • Mediana com dados não agrupados:
      • Número de termos (n) ímpar: a mediana será o elemento central dos dados da sequência, sendo sua posição dada por: (n + 1) / 2
      • Número de termos (n) par: a mediana será a média aritmética entre os dois termos centrais.
    • Mediana com dados agrupados: soma-se todos os pesos e divide por 2, a mediana será o termo central.
    • Mediana com dados agrupados em classes: mesmo método que o anterior.
  • Moda: é o valor que ocorre com maior frequência na sequência numérica.
